Credentials

Several features of this widget is inspired by various articles and code snippets that are shared by those brilliant and nice fellow developers:

  • The ripple effect on the Sidebar is inspired by Jhey Tompkins' Article.

  • The styling of the Input Button is inspired by Osvaldas Valutis' Article.

  • The color scheme is partially referencing to Chris Masterson's Dribbble post.

  • Coordinates that are used to draw arcs on the map are generated by functions developed by Baidu Map.
